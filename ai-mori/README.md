# Daymori AI Mori 模块

AI Mori 是 Daymori 的自律陪伴模块，定位为共情、坚定、可落地的成长伙伴。

## 产品底线

- 全免费、无广告、无会员、无付费引导
- 不提供作弊答案，不生成违规低俗内容
- 严重心理风险场景优先引导联系家人/老师/专业机构
- 对话最终回归现实行动，不诱导沉迷聊天

## 已实现能力

- 多轮对话 + 流式输出 + 超时重试 + 手动中断
- 本地长程记忆（加密）与偏好设置（加密）
- 目标拆解数据本地持久化（加密）
- 消息列表虚拟滚动（react-window）
- 应急关键词触发与应急行动建议
- SMART 目标拆解卡片
- 快捷设置抽屉（风格/记忆开关/应急开关/导入导出/一键清空）
- 内容安全双向过滤（输入与输出）

## 目录结构

- `src/components`: UI 原子组件
- `src/context`: Context + Reducer
- `src/hooks`: 核心业务 hooks
- `src/services`: API 与本地存储
- `src/utils`: 安全、加密、目标拆解等工具
- `src/tests`: 单元测试

## 本地运行

1. 安装依赖: `npm install`
2. 复制环境变量: `.env.example` -> `.env.local`
3. 启动开发环境: `npm run dev`
4. 运行测试: `npm run test`

## 边界说明

- 模块不做医疗、心理治疗、法律咨询替代。
- 遇到自伤/自杀倾向，固定进入危机引导文本。
- 所有持久化数据默认落地本地，不自动上传第三方。
